JARING METAL AI - BACKEND STARTUP GUIDE
========================================

1. INSTALLATION (if not already done)
   cd "/sessions/confident-funny-bardeen/mnt/Jaring Metal/JaringMetalAI/backend"
   npm install

2. DATABASE INITIALIZATION
   node database/seed.js
   
   This will create jaringmetal.json with:
   - 5 demo users with different roles
   - 3 demo customers
   - 6 material categories
   - 3 pricing formulas
   - 7 metal benchmarks
   - Realistic market prices
   - 4 sample quotations

3. START THE SERVER
   node server.js
   
   Server will run on: http://localhost:3000
   
4. TEST THE SERVER
   
   Health check:
   curl http://localhost:3000/api/health
   
   Login with demo user:
   curl -X POST http://localhost:3000/api/auth/login \
     -H "Content-Type: application/json" \
     -d '{"email":"admin@jaringmetal.com","password":"admin123"}'
   
   This returns: { "token": "...", "user": {...} }
   
   Use the token for authenticated requests:
   curl http://localhost:3000/api/customers \
     -H "Authorization: Bearer TOKEN_HERE"

7. FRONTEND INTEGRATION
   
   The server is configured to accept requests from:
   - http://localhost:5173 (Vite dev server)
   
   Connect your frontend by using:
   - Base URL: http://localhost:3000/api
   - Include JWT token in Authorization header
   - All endpoints support JSON request/response

10. TROUBLESHOOTING
    
    Port already in use:
    - Change PORT environment variable
    - Or kill existing process: killall node
    
    Database errors:
    - Delete jaringmetal.json
    - Re-run: node database/seed.js
    
    Module not found:
    - Run: npm install
    - Ensure better-sqlite3 is optional
    
    Gemini API errors:
    - Check API key in services/geminiService.js
    - Verify internet connection
    - Test with different document
